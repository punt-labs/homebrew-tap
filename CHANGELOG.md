## [Unreleased]

### Added

- quarry-menubar: add formula (v0.5.0). Installs a prebuilt, universal (arm64 + x86_64) macOS menu bar app from the quarry-menubar GitHub Release. As a formula (not a cask) the ad-hoc-signed app installs without quarantine, so it launches with no notarization or Developer ID. `brew install punt-labs/tap/quarry-menubar`.
